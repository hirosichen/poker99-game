# 撲克牌99遊戲 (Poker 99 Game)

一個使用 Next.js 構建的線上撲克牌99遊戲。

## 遊戲介紹

撲克牌99是一款有趣的卡牌遊戲，目標是第一個出完手中所有卡牌的玩家獲勝。

## 遊戲規則

### 基本規則
- 每位玩家開始時有5張牌
- 只能出與中心牌相同花色或相同點數的牌
- 第一個出完手中所有卡牌的玩家獲勝
- 當玩家出牌後，若手牌少於5張則自動補牌

### 特殊牌規則
- **9 = 99**：可以隨時打出，回到當前玩家（跳過下一位）
- **A = 1**：最小點數
- **J = 11, Q = 12, K = 13**：人頭牌

### 遊戲流程
1. 系統隨機發給每位玩家5張牌
2. 翻開第一張中心牌
3. 按順序輪流出牌或抽牌
4. 直到有玩家出完所有手牌為止

## 技術棧

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React

## 安裝與運行

1. 安裝依賴：
```bash
npm install
```

2. 啟動開發伺服器：
```bash
npm run dev
```

3. 在瀏覽器中訪問 `http://localhost:3000`
